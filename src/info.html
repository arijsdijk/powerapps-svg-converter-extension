About This Extension

PowerApps SVG Converter helps you convert SVG files into a format compatible with Microsoft PowerApps.

How to Use

Upload SVG:

  • Select an SVG file from your computer
  • Preview and converted code appear automatically
  • Click "Copy to Clipboard" to use in PowerApps

Paste SVG:

  • Switch to the "Paste SVG" tab
  • Paste your SVG code
  • Preview and converted code appear automatically
  • Click "Copy to Clipboard" to use in PowerApps

Features

  • Automatic SVG conversion
  • Live preview
  • Upload or paste SVG code
  • One-click copy to clipboard
  • Dark mode support
